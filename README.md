# 중학교 수학 정복하기 🎓

중학교 수학을 체계적으로 학습하고 연습할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 📚 **체계적인 학습**: 대수, 기하, 통계, 함수 등 중학교 수학의 모든 주제를 단계별로 학습
- ✏️ **다양한 문제**: 무작위로 생성되는 문제로 실력 향상
- 💡 **힌트 제공**: 막혔을 때 도움이 되는 힌트 제공
- 📊 **통계 기능**: 문제 풀이 통계를 통해 학습 진도 확인
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 사용 가능

## 설치 방법

1. 저장소를 클론하거나 다운로드합니다.

2. 필요한 패키지를 설치합니다:
```bash
pip install -r requirements.txt
```

## 실행 방법

애플리케이션을 실행합니다:
```bash
python app.py
```

브라우저에서 `http://localhost:5000`으로 접속합니다.

## 사용 방법

1. **홈페이지**: 메인 페이지에서 학습하고 싶은 주제를 선택합니다.
2. **학습하기**: 각 주제의 개념과 설명을 확인합니다.
3. **문제 풀기**: 무작위로 생성되는 문제를 풀고 정답을 확인합니다.
4. **힌트 보기**: 문제가 어려울 때 힌트를 확인할 수 있습니다.
5. **통계 확인**: 문제 풀이 통계를 통해 자신의 실력을 확인합니다.

## 학습 주제

### 대수 (Algebra)
- 일차방정식
- 이차방정식
- 인수분해

### 기하 (Geometry)
- 피타고라스 정리
- 원의 성질
- 도형의 닮음

### 통계 (Statistics)
- 평균, 중앙값, 최빈값
- 그래프
- 확률

### 함수 (Function)
- 일차함수
- 이차함수
- 함수의 그래프

## 기술 스택

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **스타일링**: CSS3 (반응형 디자인)
