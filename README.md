# 중학교 수학 정복하기 🎓

중학교 수학을 체계적으로 학습하고 연습할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 📚 **체계적인 학습**: 중학교 1학년부터 3학년까지 학기별(중1-1, 중1-2, 중2-1, 중2-2, 중3-1, 중3-2)로 체계적으로 학습
- 📖 **상세한 개념 설명**: 각 학기별로 교과서와 최상위 문제집의 내용을 포함한 상세한 개념 설명 제공
- ✏️ **다양한 문제**: 학기별 특성에 맞는 무작위 문제 생성으로 실력 향상
- 💡 **힌트 제공**: 막혔을 때 도움이 되는 힌트 제공
- 📊 **통계 기능**: 문제 풀이 통계를 통해 학습 진도 확인
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 사용 가능

## 설치 방법

1. 저장소를 클론하거나 다운로드합니다.

2. 필요한 패키지를 설치합니다:
```bash
pip3 install -r requirements.txt
```

## 실행 방법

애플리케이션을 실행합니다:
```bash
python3 app.py
```

브라우저에서 `http://localhost:5000`으로 접속합니다.

## 사용 방법

1. **홈페이지**: 메인 페이지에서 학습하고 싶은 주제를 선택합니다.
2. **학습하기**: 각 주제의 개념과 설명을 확인합니다.
3. **문제 풀기**: 무작위로 생성되는 문제를 풀고 정답을 확인합니다.
4. **힌트 보기**: 문제가 어려울 때 힌트를 확인할 수 있습니다.
5. **통계 확인**: 문제 풀이 통계를 통해 자신의 실력을 확인합니다.

## 학습 주제 (학기별 구성)

### 중1-1 (중학교 1학년 1학기)
- 정수와 유리수: 정수와 유리수의 개념, 절댓값, 수직선
- 정수와 유리수의 사칙연산: 덧셈, 뺄셈, 곱셈, 나눗셈, 분배법칙
- 문자와 식: 일차식, 식의 값, 동류항, 일차식의 계산
- 일차방정식: 방정식의 개념, 일차방정식의 풀이, 일차방정식의 활용
- 좌표평면과 그래프: 좌표평면, 순서쌍, 그래프, 두 점 사이의 거리

### 중1-2 (중학교 1학년 2학기)
- 기본도형: 점, 직선, 평면, 각, 평행선의 성질
- 평면도형: 다각형, 삼각형, 사각형, 원과 부채꼴, 넓이 공식
- 입체도형: 다면체, 각기둥, 각뿔, 원기둥, 원뿔, 구, 겉넓이와 부피
- 통계: 자료의 정리, 도수분포표, 히스토그램, 대표값(평균, 중앙값, 최빈값), 그래프

### 중2-1 (중학교 2학년 1학기)
- 유리수와 순환소수: 유한소수와 무한소수, 순환소수, 순환소수를 분수로 나타내기
- 식의 계산: 다항식, 일차식과 이차식, 다항식의 덧셈과 뺄셈, 곱셈과 나눗셈, 곱셈 공식
- 일차부등식: 부등식의 개념, 일차부등식의 풀이, 부등식의 성질, 일차부등식의 활용
- 연립일차방정식: 연립일차방정식의 개념, 대입법과 가감법, 해의 개수, 연립일차방정식의 활용

### 중2-2 (중학교 2학년 2학기)
- 일차함수: 함수의 개념, 일차함수의 그래프, 기울기와 절편, 일차함수의 식 구하기, 일차함수의 활용
- 도형의 닮음: 닮은 도형, 닮음비, 닮은 도형의 성질, 삼각형의 닮음 조건, 평행선과 선분의 비
- 피타고라스 정리: 피타고라스 정리와 그 증명, 피타고라스 정리의 활용, 피타고라스 수, 평면도형과 입체도형에서의 활용

### 중3-1 (중학교 3학년 1학기)
- 제곱근과 실수: 제곱근의 개념, 근호, 제곱근의 성질, 무리수, 실수, 제곱근의 사칙연산, 분모의 유리화
- 인수분해: 인수분해의 기본, 인수분해 공식, 이차식의 인수분해, 인수분해의 활용
- 이차방정식: 이차방정식의 개념, 인수분해를 이용한 풀이, 제곱근을 이용한 풀이, 완전제곱식을 이용한 풀이, 근의 공식, 이차방정식의 활용
- 이차함수: 이차함수의 개념, 이차함수 y=ax²의 그래프, 이차함수 y=ax²+bx+c의 그래프, 최댓값과 최솟값, 이차함수의 식 구하기, 이차함수와 이차방정식

### 중3-2 (중학교 3학년 2학기)
- 삼각비: 삼각비의 개념, 삼각비의 값, 삼각비의 관계, 삼각비표, 삼각비의 활용
- 원의 성질: 원과 현, 원주각, 원주각의 성질, 접선과 할선, 원의 접선의 성질, 원과 비례, 원의 방정식
- 통계: 대푯값(평균, 중앙값, 최빈값, 분산, 표준편차), 상관관계, 확률, 경우의 수, 확률의 계산, 통계적 추정

## Render 배포 가이드

이 프로젝트는 Render를 통해 쉽게 배포할 수 있습니다.

### 배포 단계

1. **GitHub에 코드 푸시**
   ```bash
   git add .
   git commit -m "Render 배포 준비"
   git push origin main
   ```

2. **Render 계정 생성 및 로그인**
   - [Render](https://render.com)에 접속하여 계정을 생성합니다.
   - GitHub 계정으로 로그인하는 것을 권장합니다.

3. **새 Web Service 생성**
   - Render 대시보드에서 "New +" 버튼을 클릭합니다.
   - "Web Service"를 선택합니다.
   - GitHub 저장소를 연결합니다.

4. **자동 설정 확인**
   - `render.yaml` 파일이 있으면 대부분의 설정이 자동으로 감지됩니다.
   - 다음 설정이 자동으로 적용됩니다:
     - **Build Command**: `pip install -r requirements.txt`
     - **Start Command**: `gunicorn app:app --bind 0.0.0.0:$PORT`
     - **Python Version**: 3.11.0

5. **수동 설정 (render.yaml이 없는 경우)**
   - **Name**: `middle-school-math` (또는 원하는 이름)
   - **Environment**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app --bind 0.0.0.0:$PORT`
   - **Plan**: Free (무료 플랜)

6. **배포 완료**
   - "Create Web Service"를 클릭하면 자동으로 빌드가 시작됩니다.
   - 빌드가 완료되면 제공된 URL로 접속할 수 있습니다.
   - 예: `https://middle-school-math.onrender.com`

### 주의사항

- Render의 무료 플랜은 15분간 요청이 없으면 서비스가 슬립 모드로 전환됩니다.
- 첫 요청 시 약간의 지연이 있을 수 있습니다 (서비스 깨우기 시간).
- 프로덕션 환경에서는 유료 플랜 사용을 권장합니다.

### 환경 변수 설정

필요한 경우 Render 대시보드에서 환경 변수를 추가할 수 있습니다:
- `PORT`: Render가 자동으로 설정합니다 (수정 불필요)
- `PYTHON_VERSION`: `render.yaml`에 이미 설정되어 있습니다
